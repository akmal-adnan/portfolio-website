## ⚡️ Getting Started

### Prerequisites

Make sure you have **Node.js** and **Yarn** installed.

### Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/akmal-adnan/portfolio-website

cd portfolio-website

yarn install
```

Run Development Server

```
yarn dev
```
